# Projetos-3

Atividade do módulo 3 de projetos — Arenafutapp

## Sobre

Este projeto contém uma aplicação dividida em backend (pasta `server`) e frontend (diretório principal).

## Pré-requisitos

- Node.js instalado
- npm instalado

## Instalação e execução

1. **Backend**
   - Acesse a pasta do servidor:
     ```bash
     cd server
     ```
   - Instale as dependências:
     ```bash
     npm install
     ```
   - Inicie o servidor:
     ```bash
     npm start
     ```

2. **Frontend**
   - Volte ao diretório principal:
     ```bash
     cd ..
     ```
   - Instale as dependências:
     ```bash
     npm install
     ```
   - Inicie a aplicação:
     ```bash
     npm start
     ```

## Observações

- Certifique-se de que o backend esteja rodando antes de iniciar o frontend.
- Para abrir a aplicação no navegador, utilize:
  ```bash
   http://localhost:3000
